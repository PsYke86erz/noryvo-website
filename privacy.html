Privacy Policy

Last updated: April 2026 · Noryvo Messenger v1.5

Data Controller: Normen Ehrentraut · Sassenberg, Germany
Contact: normen.ehrentraut@icloud.com

1. Data We Collect

Account Data: Phone number (for SMS verification via Twilio), optional username and profile picture.

Messages: Text, image, video, and voice messages stored on our servers for delivery.

Technical Data: Device type, OS version, app version, anonymized IP address (deleted after 7 days), push notification tokens.

Location (optional): GPS data only when using the "Nearby" feature. Never stored permanently. Can be revoked in device settings at any time.

2. How We Use Your Data

  • Account verification and authentication
  • Message delivery and storage
  • Push notifications
  • Optional location-based features
  • App security and abuse prevention

3. Third-Party Services

Twilio (SMS verification) – Twilio Inc., San Francisco, USA. Standard Contractual Clauses in place.

Firebase Cloud Messaging (push notifications) – Google Ireland Ltd. Only device tokens and metadata are shared, never message content.

Hetzner Online (server hosting) – Hetzner Online GmbH, Germany. All data processed exclusively in Germany.

4. Data Retention

  • Messages: until deleted by user or 730 days
  • Account data: until account deletion
  • IP addresses: 7 days

5. Your Rights (GDPR)

You have the right to access, correct, delete, restrict, or export your data. Data export is available directly in the app (Profile → Export Data). Contact us at normen.ehrentraut@icloud.com for any requests.

6. Security

TLS 1.3 encryption for all connections. Servers located in Germany. Regular security updates.

7. Children

Noryvo Messenger is not intended for users under 16. We do not knowingly collect data from children.

8. Changes

We may update this policy. Users will be notified of material changes via in-app notification.
